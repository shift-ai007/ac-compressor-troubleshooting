# Contributing

Contributions from HVAC professionals and experienced homeowners are welcome. This guide aims to be the most practical compressor troubleshooting resource available.

## How to Contribute

1. Fork this repository
2. Create a feature branch (`git checkout -b improve-electrical-section`)
3. Make your changes
4. Submit a pull request with a clear description

## Types of Contributions

- **Diagnostic techniques** — field-tested methods for identifying compressor issues
- **Pressure/temperature charts** — reference data for different refrigerant types and conditions
- **Cost data** — updated pricing for parts and labor in different regions
- **Safety corrections** — if you spot anything unsafe in our guides, please flag it immediately
- **Photo/diagram contributions** — clear images of common failure modes

## Standards

- Prioritize safety — always note when a procedure requires EPA certification or licensed technician
- Include specific measurements when possible (temperatures, pressures, voltage ranges)
- Write procedures as step-by-step lists
- Note which compressor types or brands a technique applies to
